本项目为设计模式的示例项目。

## 一. Strategy design pattern(策略模式)
1. 定义一个算法组的接口
2. 在派生类中实现具体的算法细节
3. 派生类可以使用模板方法实现
4. client通过传入的具体的Strategy实现类来执行具体的算法

Thread 类就是采用了策略模式
